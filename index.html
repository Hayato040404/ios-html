Basic Usage

iOSUI.js provides both CSS classes and JavaScript methods for creating iOS-like interfaces.

CSS Classes
<div class="ios-container"> <div class="ios-header"> <h1 class="ios-header-title">My App</h1> </div> <div class="ios-list"> <div class="ios-list-item"> <div class="ios-list-item-content"> <div class="ios-list-item-title">List Item</div> <div class="ios-list-item-subtitle">Subtitle</div> </div> <span class="ios-list-item-chevron">‚Ä∫</span> </div> </div> </div>
JavaScript API
// Show alert iosAlert('Title', 'Message'); // Show confirm dialog iosConfirm('Confirm', 'Are you sure?').then(confirmed => { if (confirmed) { console.log('User confirmed'); } }); // Show toast iosToast('Hello World!'); // Show action sheet iosActionSheet({ title: 'Choose an option', buttons: [ { text: 'Option 1' }, { text: 'Option 2' }, { text: 'Delete', style: 'destructive' } ] }).then(index => { console.log('Selected option:', index); });
