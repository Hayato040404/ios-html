🎯 Pickers

Date pickers, time pickers, and custom pickers with native iOS feel.

JavaScript
// Date picker const datePicker = iosUI.createDatePicker({ mode: 'date', value: new Date(), onChange: (date) => console.log(date) }); // Custom picker const picker = iosUI.createPicker({ columns: [['S', 'M', 'L'], ['Red', 'Blue']] });

📋 Lists

iOS-style lists with various accessories and interactive elements.

JavaScript
